# Figure 6 — Combinatorial Grammar — Redesign Design

Date: 2026-04-24
Status: design — pending implementation plan

## Problem

Current `figures/pngs/09_combinatorial_grammar.png` (renderer: `src/paper_figures/fig_combinatorial_grammar.py`) has visible polish and structural issues:

- Panel A (3×9 raw H&E grid) — tiles too small to read; all 27 tiles look visually identical, so the grid wastes space.
- Panel B (3×9 residual L2 heatmap) — title overflows past panel boundary, clips into suptitle.
- Panel C (3 stacked vertical bar charts of signed residuals) — bars cramped, in-bar value labels collide with adjacent bars and panel borders, redundant per-subplot metric ordering.

The figure also fails to land its intended message clearly. The story is: *additive (state + O2 + glucose) decomposition does not explain morphology, and the residual signal lives in cell-size metrics.* Currently neither point reads at a glance.

## Goals

1. **Replace panel A** with a per-condition pixel-difference visualization against a single reference render of the anchor's *original* (unmodified) TME — analogous to existing leave-one-out diff plots.
2. **Restructure panel B** to share a row-axis with panel A so readers map "spatial diff (A) ↔ scalar magnitude (B)" 1:1.
3. **Redesign panel C** as 3 horizontal-bar case studies (lowest / median / highest residual L2), with no in-bar labels.
4. **Move raw H&E grids to SI** — supplementary figure shows raw H&E grids for the representative anchor plus 3 additional anchors picked by sweep-response magnitude (low/mid/high tertile), to demonstrate the diff pattern is not anchor-specific.
5. **Apply Fig-5 polish pass** — L-shape outer layout, dashed panel borders, height-matched colorbars, no suptitle, inline panel titles, INK color tokens.

## Non-goals

- Re-running the A3 sweep generation. The existing 540 sweep tiles, CellViT sidecars, `morphological_signatures.csv`, and `additive_model_residuals.csv` are unchanged.
- Changing the additive decomposition model or the morphology metric set.
- Modifying CODEX-related code (per HANDOVER directive).

## Reference generation — `src/a3_combinatorial_sweep/generate_references.py`

Standalone script. Idempotent: skips anchors that already have a reference PNG at the target path.

**Behavior:**

1. Read `anchors_k20_t1_medoid.txt`.
2. For each anchor `<id>`:
   - Target path: `inference_output/paired_ablation/ablation_results/<id>/all/generated_he.png`
   - If exists → skip.
   - Else → run inference with the anchor's *original* TME channels (no sweep manipulation; all 4 groups active: `cell_types`, `cell_state`, `vasculature`, `microenv`), save PNG to target path.

**Inference reuse:** call existing inference helpers programmatically (see `tools/stage3/tile_pipeline.py` and `stage3_inference.py`). The ablation pipeline that produced the existing 5 covered references is the same code path; this script invokes the "all groups active, original channels" case for the missing 15.

**CLI:**

```bash
conda run -n pixcell python -m src.a3_combinatorial_sweep.generate_references \
  --anchors src/a3_combinatorial_sweep/anchors_k20_t1_medoid.txt \
  --output-root inference_output/paired_ablation/ablation_results \
  --device cuda
```

**Error handling:** failure on one anchor logs and continues; does not abort the batch.

## Main figure — `figures/pngs/09_combinatorial_grammar.png`

### Layout

- `figsize = (12.0, 9.0)`, white facecolor
- Outer gridspec: `2 × 2`, `width_ratios=[1.6, 1.0]`, `height_ratios=[1.0, 1.0]`, `wspace=0.22`, `hspace=0.30`
- Panel placement (L-shape):
  - `[0, 0]` → panel A (diff grid)
  - `[1, 0]` → panel B (L2 heatmap)
  - `[:, 1]` → panel C (case studies, full height)
- No suptitle. Each panel has an inline upper-left bold panel label `A`, `B`, `C`.
- Each panel has a dashed-border outer rectangle (Fig-5 9f5008d pattern).

### Panel A — `_diff_grid.py`

Renders the representative anchor's per-condition pixel-difference maps against its reference render.

**Layout inside panel A bbox** (from `subgridspec`):

- Top-left inset: 1 grid-cell-sized square showing the reference H&E. Caption immediately above the inset: `reference (original TME)` (fontsize 7, INK).
- Below the inset: 3×9 grid of difference heatmaps. Rows = states `(prolif, nonprolif, dead)`. Columns = `(oxygen, glucose) ∈ {low, mid, high}²` ordered by oxygen-major:

  ```
  cols: (low,low) (low,mid) (low,high) (mid,low) (mid,mid) (mid,high) (high,low) (high,mid) (high,high)
  ```

- Row labels (state) on left edge, fontsize 6.5.
- Column labels suppressed (shown only on panel B's bottom).
- Shared colorbar on right edge of grid, height-matched via `make_axes_locatable`.

**Per-tile diff math:**

```python
ref = imread(reference_path).astype(np.float32)            # (H, W, 3)
cond = imread(condition_path).astype(np.float32)           # (H, W, 3)
diff = np.mean(np.abs(cond - ref), axis=-1)                # (H, W)
```

**Normalization:** global per-anchor — `vmin=0`, `vmax = max(diff[s,o,g] for all 27 conditions)`. Cmap: `magma`.

**Panel label:** "A" top-left of axes bbox, bold, fontsize 13.

### Panel B — `_l2_heatmap.py`

3×9 heatmap of `residual_l2_norm` from `additive_model_residuals.csv`. Same row-axis (state) and column-axis (oxygen/glucose) as panel A.

- Cmap: `magma`. `vmin=0`, `vmax = max residual_l2_norm`.
- **Numeric L2 value rendered inside each cell.** Text color: white if `value < 0.5 * vmax`, else black (auto-contrast). Fontsize 7.
- Row labels (state) on left, fontsize 6.5.
- Column labels on bottom: `f"{o}/{g}"`, rotation 30°, ha=right, fontsize 6.
- Shared colorbar on right edge, height-matched.
- Inline title (left-aligned, fontsize 7): `interaction magnitude — residual L2 norm (sweep levels: low=0.50, mid=0.75, high=1.00)`.
- Panel label "B" top-left.

### Panel C — `_case_studies.py`

3 stacked horizontal-bar subplots inside panel C bbox.

**Case selection** (existing logic, preserved):

```python
sorted_rows = sorted(residual_rows, key=lambda r: float(r["residual_l2_norm"]))
cases = [("lowest", sorted_rows[0]),
         ("median", sorted_rows[len(sorted_rows) // 2]),
         ("highest", sorted_rows[-1])]
```

**Per-subplot rendering:**

- y-axis: 9 morphology metrics, sorted **per-subplot by |residual| descending** (each subplot has its own metric order).
- x-axis: signed residual values (numeric ticks, fontsize 6.5).
- Bars: `barh`, color `#4C78A8`, edgecolor black width 0.5.
- `axvline(0)` black width 0.8.
- y-tick labels fontsize 6.5.
- **No in-bar value labels.** Reader reads x-axis ticks instead.
- Inline subplot title (left-aligned, fontsize 7): `f"{case_label}: {state}, O2={o2}, glucose={glucose}, L2={l2:.3g}"`.
- Spines: hide top and right.

**Subplot stacking:** `subgridspec(3, 1, hspace=0.55)`. Panel label "C" top-left of the top subplot.

## SI figure — `figures/pngs/SI_09_combinatorial_grammar_anchors.png`

Renderer: `fig_combinatorial_grammar_si.py`.

**Anchor selection** (`_shared.select_si_anchors`):

1. Take the representative anchor (same one used in main panel A — `_pick_representative_anchor`).
2. Compute a per-anchor sweep-response magnitude from `morphological_signatures.csv`: for each anchor, the sum across morphology metrics of variance across that anchor's 27 sweep tiles. (This is a per-anchor proxy for "how much does morphology change across the sweep" — used because `additive_model_residuals.csv` is aggregated across anchors and does not contain per-anchor residuals.)
3. Rank the 19 non-representative anchors by this magnitude.
4. Pick `low (rank 0)`, `mid (rank 9)`, `high (rank 18)` — 3 anchors.
5. Final list: `[representative, low, mid, high]`.
6. If a picked anchor lacks a reference image (post-`generate_references.py`), skip it and fall back to the next-ranked anchor with a logged warning.

**Layout:**

- `figsize = (14.0, 10.0)`.
- 2×2 outer grid of sub-grids. Each sub-grid is the anchor's 3×9 raw H&E tile grid.
- Above each sub-grid: anchor ID + sweep magnitude (fontsize 8, INK).
- Tile borders: width 0.25, color `#8A8A8A`.
- No suptitle. Small `Figure S6` label top-left of `[0, 0]` sub-grid.
- No diffs in SI — raw H&E only.

## Error handling

| Condition | Behavior |
|---|---|
| Reference missing for representative anchor | Raise `FileNotFoundError` with hint to run `generate_references.py`. |
| Reference missing for SI anchor | Log warning, fall back to next-ranked anchor; if all candidates exhausted, render with fewer than 4 anchors. |
| Sweep tile missing for any `(anchor, state, o2, glucose)` | Raise `FileNotFoundError` with full path. |
| `morphological_signatures.csv` or `additive_model_residuals.csv` missing | Raise `FileNotFoundError` (existing behavior). |
| `generate_references.py` failure on one anchor | Log + continue; do not abort batch. |
